Greenhouse Simulator

This is a greenhouse simulator that controls temperature, moisture, and humidity.
The simulator also includes a model that simulates the impact of current weather conditions on these values.
Note: Even though applets are outdated, this was a great project to practice Java.

Description:

Real-time monitoring: Continuously tracks and displays temperature, humidity, and moisture levels.
Controls: Implements simple algorithms to adjust heating, ventilation, and humidification systems based on target conditions.
Weather simulation: Incorporates a weather model to simulate external conditions and their impact on the greenhouse environment.
Performance evaluation: Calculates and displays the rate at which controlled conditions change in response to system adjustments.
External influences: Factors in the rate at which external weather conditions affect greenhouse conditions.
Data Storage: Included an interface for storing and retrieving data.

Understanding Text Boxes in the greenHouse Applet:

Set current weather conditions: Current temperature, current humidity, and current moisture: Setting current weather conditions.
Set system parameters for furnace, air conditioner, humidifier, and sprinkler:
Set temperature, set moisture, set humidity: These are used to set targeted conditions. For example, keep the heat around a temperature of 42°C.
Temperature rate of change, moisture rate of change, humidity rate of change: These simulate the rate of change of conditions inside the greenhouse when the furnace, air conditioner, humidifier, and sprinkler are turned on. For example, when the temperature rate of change is 5°C per time unit and the current temperature is 50°C, the temperature inside the greenhouse will increase to 55°C per time unit. The time unit can be adjusted in the following section.
Temperature rate time (min), moisture rate time (min), humidity rate time (min): These specify how the rate of change per minute is adjusted. For example, we can increase the temperature by 5°C every 60 seconds or every 30 seconds.
External influences:
Set weather moisture rate change, set weather moisture rate time, set weather humidity rate change, set weather humidity rate time.
