Analyze visual data

This personal project involves extracting text-based systems of linear equations from images. It processes the images to identify and extract numerical values, builds the appropriate matrices, and solves the resulting systems of equations. It was written in C++.
Note: There is a link to the GitHub repository for this project on the top right corner.

The project involved:

vscode, c++ , APIs/Libraries : Eigen, opencv2, tesseract, leptonica, regex..., configure library paths with a Makefile, README file, and configure .json files. The goal of this project was to gain experience with various libraries and rapidly develop a prototype program.

Program Output: Most coefficients are correct but it still needs more work.
