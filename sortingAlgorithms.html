Sorting Algorithms

This is a personal project that demonstrates the implementation of various sorting algorithms using the Strategy pattern. It was written in C++.
Note: There is a link to the GitHub repository for this project on the top right corner.

The project involved:

Object-oriented programming, xcode, c++ , and using the Strategy design patterns to implement various sorting algorithms. Perhaps the most time consuming task was learning basic github commands such as staging updates, committing changes, amendding to old commits, dealing with stash, pulling, merging, .gitignore...

Here are the key characteristics of a strategy design pattern:

Family of algorithms: A related family of algorithms that solve a specific problem. I included Quick-sort, Merge-sort, Heap-sort, Insertion-sort, and the infamous Bubble-sort .
Encapsulation: Each sorting algorithm is encapsulated in a separate class, making them independent.
Interchangeability: New classes for different sorting algorithms can be easily added and tested.
